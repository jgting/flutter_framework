# l10n
1.建立或更新arb
ex:app_zh.arb
{
"@@locale": "zh",
"hello": "哈囉",
"changeLanguage": "切換語言",
"darkMode": "深色模式"
}

2.cmd輸入flutter gen-l10n
產出新的l10n.dart

3.直接使用AppLocalizations.of(context).hello

# 語言／地區	    .arb 檔名	                @@locale值	
英文（預設）	    app_en.arb	                "en"	    
簡體中文	        app_zh.arb 或 app_zh_CN.arb	"zh" 或 "zh_CN"
